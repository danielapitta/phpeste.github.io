Financiamento coletivo

Serão 2 dias de muito aprendizado, muita mão na massa e, principalmente, de gente "arretada da peste" que irá ampliar ainda mais seu networking. Com a participação de palestrantes internacionais e grandes nomes nacionais, teremos palestras, minicursos e hackathons focados na temática do evento: crescimento e colaboração, ou seja, não será apenas um evento no qual você ganhará um certificado para guardar na gaveta, você estará desenvolvendo seu currículo e, ao mesmo tempo, desenvolvendo o ambiente do código aberto.

Palestras

 

Dia 03

Horário Auditório 1 Auditório 2 Auditório 3
09:00 Credenciamento
10:00 Keynote - Marco Pivetta
11:00 Proteja sua aplicação com o Zend Framework 2
Cyrille Grandval

Os frameworks contêm um conjunto de ferramentas que permitem proteger de forma rápida e eficiente uma aplicação durante o seu desenvolvimento. No entanto, nem tudo é automático e é necessário que o desenvolvedor as conheça e consiga usá-las da melhor forma. A palestra irá apresentar as diversas funcionalidades disponibilizadas pelo Zend Framework 2 para construir uma aplicação PHP segura e também uma ferramenta de encoding e ofuscação de código-fonte.		 Deploy Inteligente
Jonhnatha Trigueiro

Irei falar sobre um pouco de DEVOps para PHP. O publico alvo são pessoas que trabalham há um certo tempo com a linguagem e que querem melhorar o processo de trabalho para utilizar de metodologias ágeis, reduzindo o tempo de deploy em mais de 95%!		 Alternativas de Bancos de Dados NoSQL
Franklin Rabay

Firebase, CouchDB, PouchDB, são novas alternativas de Banco de Dados, levando o desenvolvimento de sistemas a um nível mais alto. CouchDB + PouchDB + MySQL = Nunca mais precise esperar sua internet voltar para utilizar um sistema WEB (Ou Mobile). Com essa química poderosa é possível criar aplicações dentro das especificações do manifesto Offline First!
12:00 Almoço/Desconferência
14:00 Criando aplicações stateless no PHP, vantagens e desvantagens
Rodrigo Ibraim

A ideia desse tema, surgiu junto com o conceito de PaaS (plataforma como serviço). Muitos desenvolvedores já me perguntaram como isso funciona, vantagens e desvantagens, até onde podemos ir, limitações e como encontrar soluções. Vou falar também sobre a união de soluções que vieram para agregar valor ao processo de desenvolvimento, como git docker e outros.		 PHP para Hackers, enganando sistemas para não ser enganado
Lenon Leite

Enganar sistemas para não sermos enganados? Todos devem estar pensando, "como assim?" Como proteger de algo que não conhecemos? Como proteger de ataques que nem ao menos sabemos como funcionam?! É simples, vamos aprender atacando. Nessa apresentação vou tentar passar por todas as etapas do processo de desenvolvimento em um projeto em PHP, e demonstrar que temos pontos críticos de segurança do inicio até o processo de deploy, irei analisar as partes mais sensíveis com ataques bem sucedidos, e tudo isso de forma real.		 Spaghetti com amêndoas: desenrolando seu código Javascript
Arthur Gouveia

Uma breve introdução à Module Pattern do Javascript, aliado à Almond.js (AMD) e bom senso. Aprenda a construir um code base bem estruturado e reutilizável para suas aplicações web, ao invés do spaghetti code usual.
15:00 Open-source all the things
Jefersson Nathan

Hoje em dia open-source está na moda, mas como usar essa poderosa rede de comunicação e iteração que liga pessoas ao redor do mundo ao seu favor? Saiba como usar open-source para alavancar sua carreira, e enxergue as comunidades com outros olhos.		 PHP para além dos sites
Marianna Teixeira

Nessa palestra você vai descobrir do que o php é capaz, além de dinamizar muito bem os sites por aí, com cases e exemplos práticos. De iniciantes a veteranos, o único requisito é querer experimentar!		 Criando aplicações desktop com Javascript
Sérgio Vilar

Aprenda a criar aplicações Desktop usando io.js e tudo que pode ser renderizado pelo browser: HTML, CSS e Javascript! Esta palestra vai apresentar desafios no desenvolvimento desktop e como fazê-lo com o Electron, framework usado na criação do Atom, Slack e Visual Studio Code.
16:00 Coffee/Desconferência
17:00 Quem disse que elePHPante não voa?
Jonata Weber

Serão apresentadas algumas práticas que tornam sua aplicação PHP horizontalmente escalável. Além de mostrar como o Google App Engine pode lhe ajudar nessa tarefa.		 JWT - To authentication and beyond!
Luís Cobucci

Utilizamos tokens para identificar recursos e tentar garantir segurança dos dados em ambientes inseguros. Que tal conhecer uma maneira simples e eficaz de realizar essa tarefa, garantindo a segurança e integridade dos dados? Bora entender o mundo mágico dos Json Web Tokens!		 SUIT CSS como padrão de escrita de estilos
João Paulo Barbosa Neto

A intenção da palestra é dar uma visão geral sobre alguns padrões de escrita de estilos, destacar seus benefícios, aprofundar nas convenções de nomenclatura do SUIT CSS e mostrar através de exemplos porque o adotei na minha empresa.

 

Dia 04

Horário Auditório 1 Auditório 2
09:00 Café da manhã / Desconferência
10:00 Guia do mochileiro para escalabilidade
Augusto Pascutti

Escalabilidade em um sistema é planejada desde a sua concepção ou as consequências deverão ser aceitas. Como entender a arquitetura de uma aplicação web, como usar essa arquitetura a seu favor e quais são as práticas mais comuns será o conteúdo dessa palestra. Nada de teoria, você leva pra casa um conjunto de VMs pra experimentar e aprender o que foi apresentado.
11:00 Silex 101
Nanderson Sousa de Castro

Nesta palestra vamos demostrar o poder do silex na criação de apps. Do 0 à API. Mostrando todas as features que esse microframework nos propõe em seu leque de ferramentas.		 Hacker do bem, quebrando as principais dicas de desenvolvimento seguro
Thiago Dieb

O objetivo da palestra é demonstrar como é possível encontrar tutorias e dicas em sites conhecidos e famosos trazendo informações incompletas e algumas incoerentes sobre desenvolvimento seguro, influenciando desenvolvedores na criação de códigos vulneráveis. A partir desses exemplos será possível indicar as correções e provar como é fácil criar um código com falhas e se tornar mais uma vítima.
12:00 Programação Funcional em PHP: saia da zona de conforto
Marcel dos Santos

Nesta apresentação será abordado paradigmas de programação, programação funcional e conceitos relacionados (funções de primeira classe, funções de alta ordem, funções puras, estados, imutabilidade e recursão), programação funcional em PHP, lazy evaluation, map, reduce e filter, funções parciais, memoization e currying, bibliotecas e projetos em PHP com foco em programação funcional, vantagens e desvantagens do paradigma, comparativos com outras linguagens e o futuro do PHP em relação a programação funcional.		 Criando APIs usando o micro-framework Respect
Ivan Rosolen

Como criar um API Restfull rapidamente usando componentes do micro-framework para agilizar o desenvolvimento e garantir as informações. Componentes abordados na palestra (Validation, Router, Relational e Config). Criando uma API para cadastro de cervejas (RestBeer). Código da API no github e como instalar utilizando composer e vagrant
13:00 Almoço/Desconferência
14:00 PHP 7, Ao Infinito e Além
Rafael Queiroz

PHP 7 vai chegar mais cedo do que você pensa. Nesta palestra irei abordar novas funcionalidades avançadas introduzidas no PHP 7, compatibilidade com versões anteriores, características de alto impacto e as novas interfaces de mensagens HTTP (PSR-7).		 Agrosmart: tecnologia enfrentando o desperdício de água
Anderson Casimiro

A Agrosmart é uma startup que busca reduzir drasticamente o desperdício de água e melhorar a produtividade de plantios mundo afora. Nossa arquitetura é baseada nas melhores práticas em desenvolvimento PHP, desde o campo até nossos clientes. Como fazer o PHP comunicar-se com nosso hardware, consolidar informação e distribuir... arquitetura de microsserviços e usando muito componentes PHP.
15:00 Testando sua aplicação PHP através de TDD e BDD
Daniela Pitta

De que forma você testa as suas aplicações? Nessa palestra irei falar sobre os conceitos existentes para teste e também demonstrar via código PHP como utilizar esses conceitos. Essa palestra é destinada tanto para os iniciantes em PHP quanto para os mais experientes.		 Os 15 Mandamentos do Bom Programador
Paulo Eduardo

Palestra aborda 15 (talvez mais) dicas importantes que muitas vezes são conhecidas pelos desenvolvedores de diferentes linguagens mas esquecidas, e que podem auxiliar bastante o processo de desenvolvimento e melhorar a produtividade do profissional.
